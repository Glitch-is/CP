CP
==

Competitive Programming Library for Python 3

## Usage
    import cp

## Functions
    Primes:
        isPrime(n)
        sieve(n)
        
    # GCD:
        gcd(a, b)
        lcm(a, b)
        
    Fibonacci:
        fib(n)
        
    Fractions:
        multiplyFractions(a, b)
        addFractions(a, b)
        reduceFraction(a)

    Documentation can be located in the function's implmentation
